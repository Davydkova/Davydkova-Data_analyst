# Проект 1: Анализ выгодного тарифа для телеком компании 

Проведен предварительный анализ использования тарифов на выборке клиентов,проанализировано поведение клиентов при использовании услуг оператора ирекомендованы оптимальные наборы услуг для пользователей. Проведена предобработкаданных, их анализ. Проверены гипотезы о различии выручки абонентов разных тарифов и
различии выручки абонентов из Москвы и других регионов.
